# Coming Balik — A Returning Doctor's Guide

A free, single-page reference for Malaysian doctors in the UK weighing up — or already planning — a move back home. Covers MMC & NSR registration, the Returning Expert Programme, the job market (public vs. private), family logistics, and the first 90 days after touchdown. Includes a working checklist that saves your progress in your own browser.

This isn't legal or immigration advice, and it isn't a recruitment service — for hands-on help with job matching, see [MatchSync](https://www.matchsync.com). For community discussion, see the [Dobbs forum](https://www.facebook.com/dobbsforum/). This page is just the paperwork laid out in one place, with links to the official sources.

## Using this page

Open `index.html` in any browser — no install, no build step. It's a single static file with no dependencies beyond a Google Fonts link.

## Keeping it current

The facts here (MMC/NSR process, REP eligibility, tax treatment) are accurate as of June 2026 but **will** drift — these rules have changed before and will again. If you spot something outdated:

1. Edit `index.html` directly (it's plain HTML/CSS/JS — search for the relevant section text and update it).
2. Add a line to the note boxes (`<div class="note">`) flagging what changed and when, rather than silently overwriting — it helps the next reader trust the page.
3. Commit the change through GitHub's web UI (no command line needed): open the file in the repo, click the pencil icon to edit, then **Commit changes**.

## Disclaimer

Always confirm anything with a hard deadline (REP applications, APC renewals) directly with MMC, NSR, TalentCorp, or LHDN. This page is an orientation, not a substitute for the official process.
